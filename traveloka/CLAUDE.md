## Repository Overview

This is a data repository, not a software project. It contains a single JSON file (`地区.json`) with a Traveloka API response snapshot — geographic region and hotel search data for European locations (Valencia, Lyon, Toulouse, Strasbourg, etc.).

There are no build commands, tests, linting, or application code. The JSON structure follows Traveloka's search API response format with fields like `geoRegionContent`, `autoCompleteContent`, `hotelContent`, etc.
